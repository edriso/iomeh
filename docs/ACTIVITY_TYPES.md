# Activity Types Documentation

## Overview
This document describes the 75 activity types available in the IOMeH health tracking platform. Activities are organized into four main categories: Workout, Nutrition, Wellness, and Mindfulness.

## Point System

### Base Points Guide
- **Low Intensity (10-20 pts)**: Light activities, easy daily habits
- **Moderate Intensity (25-35 pts)**: Regular daily activities requiring effort
- **High Intensity (40-50 pts)**: Vigorous activities, major commitments
- **Very High Intensity (55-60 pts)**: Extreme effort, exceptional activities

### Daily Target
The suggested daily target is **90-100 points**, achieved through:
- **Workout**: 30-45 pts (1-2 activities)
- **Nutrition**: 25-35 pts (meals + hydration)
- **Wellness**: 20-30 pts (sleep + recovery)
- **Mindfulness**: 15-25 pts (1-2 practices)

## Design Principles

### Simple and Clear
- Activity names are straightforward and easy to understand
- Descriptions clearly explain what's required
- No complex jargon or technical terms

### Culturally Appropriate
- Respects Islamic values and practices
- Includes Islamic spiritual activities (prayer, Quran, dhikr)
- Avoids prohibited activities (music, etc.)

### Evidence-Based
- Point values reflect actual effort and health benefits
- Based on general fitness and health guidelines
- Encourages balanced daily routine across all categories

### Inclusive
- Activities suitable for different fitness levels
- Mix of easy and challenging options
- Accessible to global users with various lifestyles

## Technical Implementation

### Database Schema
```sql
activity_types
├─ id (primary key)
├─ name (unique)
├─ category (enum: workout, nutrition, wellness, mindfulness)
├─ base_points (integer)
├─ icon (emoji string)
├─ display_order (integer)
├─ description (text)
├─ is_active (boolean)
└─ timestamps
```

### Note on MET Values
The `met_value` column has been removed from the system as it was not being used in any calculations. The application only uses `base_points` for scoring activities.

## Modification Guide

To add or modify activities:

1. Edit `/database/seeders/ActivityTypeSeeder.php`
2. Follow the existing structure for consistency
3. Assign appropriate base points (10-60 range)
4. Choose relevant emoji icon
5. Set proper display_order for sorting
6. Write clear, simple description
7. Run migrations and seeder: `php artisan migrate:fresh --seed`
